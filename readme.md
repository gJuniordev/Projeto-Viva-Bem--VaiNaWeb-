O **Viver Bem** é uma landing page focada na conscientização sobre a saúde preventiva. O projeto foi criado para informar de maneira clara e visual quais exames de rotina são necessários em cada fase da vida, separando as necessidades dos públicos masculino e feminino.

---

## 📱 Visualização do Projeto

O site foi desenvolvido com foco na **experiência do usuário (UX)** e segue princípios de **Mobile First**, garantindo que as informações estejam acessíveis em:
* Desktops e Notebooks
* Tablets
* Smartphones (Layout totalmente adaptável)

---

## 🚀 Tecnologias Utilizadas

Para a construção deste projeto, utilizei as seguintes tecnologias:

* **HTML5:** Estrutura semântica para melhor acessibilidade e SEO.
* **SASS (Syntactically Awesome Style Sheets):** Utilizado para organizar o CSS através de variáveis, aninhamento e uma arquitetura mais limpa.
* **CSS3 (Flexbox):** Para garantir o alinhamento dinâmico dos cards e seções.
* **Google Fonts:** Tipografia 'Poppins' para garantir uma leitura moderna e confortável.

---

## ✨ Funcionalidades

- [x] **Navegação Inteligente:** Menu fixo com âncoras para seções específicas.

## 🛠️ Como rodar o projeto localmente

1.  Clone este repositório:
    ```bash
    git clone https://github.com/gJuniordev/Projeto-Viva-Bem--VaiNaWeb-.git
    ```
2.  Navegue até a pasta do projeto:
    ```bash
    cd plataforma-viver-bem
    ```
3.  Caso queira editar o estilo, certifique-se de ter o SASS instalado e execute o monitoramento:
    ```bash
    sass --watch style.scss style.css
    ```
4.  Abra o arquivo `index.html` no seu navegador (ou use a extensão **Live Server** do VS Code).
